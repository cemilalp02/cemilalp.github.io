AI-Powered Code Review Bot

This project implements an automated code review assistant using GPT-based AI models. It integrates with GitHub via webhooks to analyze pull requests and provide feedback.

Features

  • Automatic analysis of coding standards
  • Identification of potential bugs and security issues
  • Inline comments on pull requests
  • Customizable rule set

Technologies Used

  • Python — Data processing and rule enforcement
  • Node.js — GitHub webhook server
  • OpenAI GPT API — Natural language analysis
  • HTML/CSS — Pull request comment integration UI
